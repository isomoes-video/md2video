中文标题：Claude Code 质量复盘
English Title: Claude Code Quality Postmortem

这个视频解读 Anthropic 在 2026 年 4 月 23 日发布的 Claude Code 质量复盘，说明近期用户感受到的质量下降并不是底层 API 或推理层整体退化，而是三个产品层变更叠加造成的体验问题。视频按时间线梳理默认推理强度下调、空闲会话思考清理 bug、以及限制输出长度的系统提示词变更，解释它们如何分别影响用户对智能、记忆和编码质量的感受。

视频最后总结 Anthropic 的改进措施，包括更多内部人员使用公开版本、更强的代码审查上下文、更严格的提示词变更评测与审计，以及对可能影响智能的变更采用更谨慎的观察期和渐进发布。核心结论是，用户感受到的模型质量是整个系统共同产生的：模型行为、产品默认值、上下文管理、提示词、评测覆盖和发布流程都会共同塑造最终体验。
